This is a mock site to display restaurant reviews.


Availables paths include:

GET -> /
GET/POST -> /places
GET -> /places/new
GET/PUT/DELETE -> /places:id
GET -> /places/:id/edit
POST -> /places/:id/rant
DELETE -> /places/:id/rant/:randId


Your places should have:

name (string)
city (string)
state (string)
cuisines (string)
pic (string)
